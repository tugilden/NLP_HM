# Многоязычный перевод на основе векторных представлений слов

## Описание проекта

Этот проект демонстрирует систему машинного перевода между украинским и русским языками, основанную на выравнивании векторных пространств.

## Ключевая идея

Если два языка имеют схожую структуру и лексику (как украинский и русский), то пространства их векторных представлений слов могут быть выровнены с помощью линейного преобразования. Мы обучаем такое преобразование на небольшом словаре пар слов, а затем используем для перевода.

##  Техническая реализация

### Основные компоненты:

1. **Векторные представления слов**
   - Используются предобученные FastText эмбеддинги (300 измерений)
   - Украинский: `cc.uk.300.vec`
   - Русский: `cc.ru.300.vec`

2. **Линейное преобразование пространств**
   - находим оптимальную матрицу W
   - Минимизируем расстояние между соответствующими векторами

3. **Два подхода:**
   - **Простая линейная регрессия** - базовый метод
   - **Ортогональное преобразование** - сохраняет структуру пространства

###  Quickstart

```sh
pip install virtualenv
virtualenv env
./env/Scripts/activate.sh
pip install -r requirements.txt
```
